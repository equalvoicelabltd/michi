# 📖 Michi 內容策略指南

## 第一部分：品牌內容框架

### 核心故事線

```
Michi 的三個核心內容支柱：

1️⃣ 信任之旅 (Trust Journey)
   └─ 展示 Michi 如何帶領用戶找到真正值得的商品
   └─ 內容: 買家故事、代購商推薦、成功案例

2️⃣ 日本發現 (Japan Discovery)
   └─ Michi 是發現日本最新、最好商品的地方
   └─ 內容: 新品介紹、商品故事、日本文化背景

3️⃣ 社區連接 (Community Connection)
   └─ 買家和賣家在 Michi 建立信任關係
   └─ 內容: 對話故事、用戶評價、社群互動
```

## 第四部分：多語言內容指南

### 繁體中文 (香港/台灣)

```
品牌語言:
├─ 溫暖親切，故事感強
├─ 用「你」而不是「用戶」
├─ 強調「信任」和「指引」
└─ 常用詞彙: 之道、之旅、精選、發現

範例:
"Michi 帶你找到最值得的日本商品"
"開始你的 Michi 之旅"
"Michi 精選代購商"
```

### 簡體中文 (馬來西亞/新加坡)

```
品牌語言:
├─ 稍微正式一些，但仍溫暖
├─ 用「您」在特定場景
├─ 強調「發現」和「品質」
└─ 常用詞彙: 之道、旅程、精选、发现

範例:
"Michi 帮您发现最好的日本商品"
"开始您的 Michi 之旅"
"Michi 精选代购商"
```

### 英文

```
品牌語言:
├─ 簡潔優雅，故事感足
├─ 多用比喻 (path, journey, guide)
├─ 強調 trust 和 discovery
└─ 常用詞彙: path, journey, guide, discover

範例:
"Michi - Your Path to Japan's Best"
"Start Your Michi Journey"
"Michi's Curated Buyers"
```

### 日文

```
品牌語言:
├─ 回歸 Michi 原義
├─ 「道」の哲学を強調
├─ 敬意と親切さを表現
└─ 常用詞彙: みち、導く、発見、信頼

範例:
"Michi - あなたの日本代購の道"
"Michi の旅を始める"
"Michi が選ぶ代購商"
```

### 泰文

```
品牌語言:
├─ 溫暖親切，符合泰國文化
├─ 強調「信任」和「幫助」
├─ 尊重傳統，接納現代
└─ 常用詞彙: ทาง、การค้นหา、เลือก、เชื่อถือ

範例:
"Michi - ทางของคุณไปยังสินค้าญี่ปุ่นที่ดีที่สุด"
"เริ่มต้นการเดินทาง Michi ของคุณ"
"ผู้ซื้อที่ Michi เลือก"
```
